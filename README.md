# Парсер сайта с рецептами
_________

Парсер выводив в файл excel список категорий блюд и ссылки на них.

**В ходе работы использовались:**
- модуль *request*
- модуль *openpyxl*
- библиотека *beautifulsoup*
- библиотека *pandas*
