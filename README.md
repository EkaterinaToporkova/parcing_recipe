# Парсер сайта с рецептами
_________

Парсер выводит в файл Excel список категорий блюд и ссылки на них.

**В ходе работы использовались:**
- модуль *request*
- модуль *openpyxl*
- библиотека *beautifulsoup*
- библиотека *pandas*
- библиотека *seaborn*
